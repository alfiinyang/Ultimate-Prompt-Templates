# Career Planner Prompt Templates

This repository contains prompt templates designed to customize a Career Planner LLM.

## File Overview

* **`data-collection-skill.md`**: Defines the specific workflow the model will follow.
* **`instructions.txt`**: Provides the essential context the model needs to apply the workflow correctly.

## Setup Instructions

1.  **Upload Knowledge Base**: Download `data-collection-skill.md` and upload it to the knowledge/files section of your custom model:
    * **Google Gemini**: "Knowledge"
    * **ChatGPT**: "Project Sources"
    * **Claude AI**: "Project Files"
2.  **Configure Instructions**: Copy the full text from `instructions.txt` and paste it into the **Instructions** or **System Prompt** text area of your custom model.

## Customization

You are encouraged to edit both `instructions.txt` and `data-collection-skill.md` to better suit your specific career planning use case or professional requirements.
