# Ultimate Slide Builder Prompt Templates

This repository contains the ultimate prompt templates designed to customize the Ultimate Slide Builder LLM for creating professional and educational presentations.

## File Overview

* **`slide-builder-template.md`**: Defines the specific workflow the model follows to structure and design slide content.
* **`instructions.txt`**: Provides the essential context and system persona the model uses to apply the slide-building workflow.

## Setup Instructions

1.  **Knowledge Base**: Download `slide-builder-template.md` and upload it to the knowledge/files section of your custom model:
    * **Google Gemini**: "Knowledge"
    * **ChatGPT**: "Project Sources"
    * **Claude AI**: "Project Files"
2.  **System Instructions**: Copy the contents of `instructions.txt` and paste them into the **Instructions** or **System Prompt** text area of your custom model.

## Customization

You can edit both `instructions.txt` and `slide-builder-template.md` to suit your specific presentation style, branding requirements, or educational use case.
