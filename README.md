# Welcome to the Ultimate Prompt Templates Library! 👋

I’m so glad you’re here! This repository is a collection of tested custom prompts designed to help you get much better responses from your AI models across a variety of tasks. 

**I thought to share some of the prompts that help me stay productive**, and I hope they help you level up your workflow too! 🚀

## 📂 How to Navigate this Repo

It’s pretty simple: each folder contains a specific tool or workflow. To get started:
1.  Browse the folders below.
2.  **Read the README file** inside each specific folder to understand what that prompt does and how to set it up in your favorite AI tool (like Gemini, ChatGPT, or Claude).
